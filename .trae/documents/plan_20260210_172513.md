Untuk mendukung file PDF, kita perlu menambahkan langkah "konversi" di browser menggunakan library **PDF.js**. Library ini akan merender halaman PDF menjadi gambar (canvas), yang kemudian bisa dibaca oleh Tesseract.js.

Berikut adalah rencana implementasinya:

### 1. Update UI & Input
- Ubah teks instruksi menjadi "Drag & Drop a Purchase Order (Image/PDF)".
- Update atribut `accept` pada `<input type="file">` agar menerima `.pdf`.

### 2. Integrasi PDF.js
- Tambahkan logika untuk memuat library `pdf.js` dari CDN secara dinamis (seperti Tesseract).
- Konfigurasi `GlobalWorkerOptions.workerSrc` agar PDF.js bisa berjalan.

### 3. Logika Proses PDF
- Update fungsi `validateAndProcess`:
  - Jika **Image**: Lanjut ke `processFile` (seperti sekarang).
  - Jika **PDF**: Panggil fungsi baru `processPdf`.
- Buat fungsi `processPdf(file)`:
  - Baca file PDF menggunakan `pdfjsLib`.
  - Render halaman pertama PDF ke dalam elemen `<canvas>` tersembunyi.
  - Konversi hasil render canvas menjadi gambar (Data URL).
  - Oper gambar tersebut ke fungsi `processFile` untuk di-OCR oleh Tesseract.

Dengan cara ini, sistem akan otomatis mengubah PDF menjadi gambar di balik layar sebelum melakukan scan teks.
